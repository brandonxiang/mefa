# mefa (Developing)

A meta framework for micro-frontend, which is inspired by [Mooa]() and [Single-SPA]().

## Usage

#### Portal System

```javascript
import { mefa } from 'mefa'
```

#### SubSystem

```javascript
import { appMefa } from 'subMefa' 
```
